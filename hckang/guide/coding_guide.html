웹 스타일 가이드 정의

본 프로젝트는 모바일/PC 반응형으로 퍼블진행되었으며 페이지 생성 및 유지보수 진행시 화면 레이아웃은 하단 코드를 참고하여 진행한다.
javascript파일은 css 보다 하단에 위치한다.

주석처리

마크업진행시 주석처리 정의 : 주석은 HTML 작업시 해당 section 구분마다 주석처리를 해주며, 수정및 ISSUE발생시 주석처리를 한다.
주석 문구의 경우 특별한 정의는 없으며, 주석처리시 해당하는 부분에 시작과 종료를 함께 작성 및 수정및 ISSUE 주석의 경우 날짜 기입을 원칙으로 한다.
종료 주석은 텍스트 앞에 //를 붙여 작성한다.
