Create New Library Content Page

아래 양식을 작성해 신규 라이브러리 콘텐츠를 업로드 하십시오.
제출이 완료되면 관리자가 피드백을 전달해 드립니다.

  • 라이브러리 콘텐츠
    정보 입력

  • Media, Documents,
    상세 정보 입력

  • 리뷰 등록 완료

라이브러리 콘텐츠 파일 등록

라이브러리 콘텐츠 입력 시 사용할 Media(Image, Video, Sound)와 BMS 사용자에게 소개할 프로젝트 Documents를 일괄 등록해주세요.

0개, 0 byte 추가됨
파일 이름 진행률 파일크기

이곳을 더블클릭 또는 파일을 드래그하세요.

첨부파일은 총 10GB를 초과할 수 없습니다.

Image(jpg, gif, png, tiff, raw), Video(mp4, avi, wmv, mkv, mov, webm), Sound(mp3), Documents(pdf, docx, pptx, excel, txt),
압축파일(zip, 7z) 형식만 업로드 할 수 있습니다.

라이브러리 콘텐츠 리뷰 스텝별 가이드

  • Step 1. 라이브러리 콘텐츠 정보 입력 - 파일 등록

    라이브러리 콘텐츠 입력 시 사용할 Media, BMS 사용자에게 소개할 모든 파일을
    등록해주세요. 등록한 파일을 사용하여 라이브러리 콘텐츠를 입력합니다.

  • Step 1. 라이브러리 콘텐츠 정보 입력 - Editor

    등록한 Media를 활용하여 라이브러리 콘텐츠 화면을​ 구성해주세요.
    Editor에 입력한 내용 그대로 화면에 노출됩니다.

  • Step 2. Media, Documents 상세 정보 입력​

    Step1에서 등록한 Media, Documents의 ​상세 정보를 입력해주세요.
    BMS 사용자에게 고지할​ 파일의 상세 정보와 저작권 정보를 입력합니다.

  • 파일 등록 가이드

    • 더블클릭 또는 파일을 드래그하여 BMS 사용자에게 소개할 프로젝트
      Media, Documents을 일괄 등록해주세요.
    • Image(jpg, gif, png, tiff, raw), Video(mp4, avi, wmv, mkv, mov,
      webm), Sound(mp3), Documents(pdf, docx, pptx, xlsx, txt), 압축
      파일(zip)을 업로드할 수 있습니다.
    • Editor로 콘텐츠 입력 시, zip파일 내에 있는 미디어 파일은 사용할 수 없으
      니, 파일을 개별로 업로드해주세요.​
    • 첨부파일은 총 10GB를 초과할 수 없습니다.

    Editor 가이드

    • Editor 상단에 있는 버튼을 클릭하여 텍스트, 이미지, 비디오, 표 등으로 라
      이브러리 콘텐츠를 입력하세요.

    • 폰트 종류, 크기, 스타일, 컬러 설정

    • 텍스트 Align, bullet, numbering 설정

    • 표, 이미지, 비디오

    • Editor 확대, HTML 코드뷰

  • Media 카테고리 설정 가이드​

    • Media를 그룹핑할 때 사용합니다.
    • 예를 들어 Kick-Off Films로 그룹핑하여 해당 카테고리에 속하는 Media를 구분할 수 있습니다.

    Documents 카테고리 설정 가이드

    • Documents는 저작권 정보 별로 그룹핑할 수 있습니다.
    • 카테고리 별 저작권 정보를 입력하거나, 모든 Documents 일괄 저작권 정보를 입력하여 사용합니다.
