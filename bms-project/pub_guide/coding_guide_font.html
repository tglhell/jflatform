FONT

◎ 폰트 정의

- 언어별 폰트 정의

font-weight: 400 - "HyundaiSansHeadKR - Regular",
font-weight: 500 - "HyundaiSansHeadKR - Medium",
font-weight: 400 - "HyundaiSansTextKR - Regular",
HyundaiSansTextKR는 클래스 "htr"를 추가한 후 font-weight를 조절함.

[HyundaiSansHeadKR - Regular] 이 텍스트는 샘플 텍스트입니다.

[HyundaiSansHeadKR - Medium] This text is sample text.

[HyundaiSansTextKR - Regular] 이 텍스트는 123456 This text is sample text. 샘플 텍스트입니다.
