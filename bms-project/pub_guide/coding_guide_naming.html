네임 정의

html 파일명 - 단어와 단어 사이에 "_" 언더바 사용, 화면내용을 암시할 수 있도록 네이밍한다. EX) upload_list.html

class - 단어와 단어사이에 "-" 하이픈 사용, 숫자는 하이픈 없이 기본두자리수로 작성, 경우에따라 1은 생략할 수 도 있다. EX) btn-area, table-type, table-type02

id - 기본적으로 camelCase사용한다. EX)gnbToggle, searchEvent

image - 언더스코어 "_"로 정의 한다. 숫자는 하이픈 없이 기본두자리수로 작성, 경우에따라 01은 생략할 수 도 있다.
