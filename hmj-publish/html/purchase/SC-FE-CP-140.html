金融/その他

  • Hyundai自動車専用カードローンチプロモーション
    30万 (15万割引 + 15万キャッシュバック)
  • モビリティプラン

    モビリティプラン

    - 対象 : 個人、個人事業者

    [注意事項]
    1. モビリティプランは、Hyundai自動車専用カード(モビリティ/EVカード)、もしくは HyundaiカードMで、頭金として10%以上を支払う場合、金利引き下げを適用する商品
    2. ローン期間は、最大36/48/60カ月まで可能
    3. その他の金融プロモーションと重複適用不可
    2.7% (モビリティ標準タイプ)
  • N-FINANCE ローンチ低金利ローンプロモーション

    N-FINANCEローンチ低金利ローンプロモーション(猶予自主返済型)

    - 対象 : 個人、個人事業者
    - 特典 : 猶予自主返済型の低金利ローン(金利-一般: 3.3%/モビリティ: 2.7%)+N顧客スペシャル付加サービス25%割引
    - 車両 : 当月生産を含む

    [ご案内]
    1. 猶予自主返済型の低金利ローンは、36カ月まで最大80%の猶予を選択できます。
    2. 中途返済手数料が免除され、ローン期間中に追加費用なしで自由に元金を返済可能
    3, モビリティローンは、Hyundai自動車専用カード(モビリティ/EVカード)、もしくはHyundaiカードMで頭金として1%以上を支払いの際、金利引き下げを提供
    4. N顧客スペシャル付加サービスは、インジェ・スピーディウムNカード、N専用洗車用品(2万ウォン追加割引)、HyundaiセレクションスペシャルNパック25%割引バウチャーを提供(詳細は、Hyundai自動車の販売店にお問合せ)

    [注意事項]
    1. ローン期間は、36カ月まで
    2. 低金利ローンと重複適用はできません 。(モビリティローンは重複適用できます。)
    2.7% (モビリティ猶予自主返済タイプ)

その他の特典

  • ブルーメンバースポイント先使用

    Blueメンバースポイント先使用

    - 対象 : 個人、個人事業者中、最低２回以上再購入した顧客
    - 特典 : 購入回数 2回-10万、3回-20万or15万、4回-30万or25万、5回-40万or35万 ポイント提供予定、ブルーメンバースポイントの先使用

    [注意事項]
    1. 提供予定のポイントが先使用ポイントより少ない場合は、使用不可
    40万
  • 展示車購入

    展示車購入条件

    - 対象 : 個人、個人事業者、法人直購入顧客
    - 車両 : 2021年5月31日以前入庫限定

    [注意事項]
    1. 購入可能な展示車の在庫がある場合にのみ可能な割引であるので、支店/代理店にお問合せください。
    2. その他の割引と重複適用ができない場合があります。
    20万
  • セーブオート

    Hyundaiカードセーブオート

    車両購入の際、最大50万ウォンまで割引を適用し、M系のHyundaiカードをご利用の際、毎月貯まるMポイントで最長36カ月間、返済するポイントを先に割引するプログラム
    - 対象 :HyundaiカードM系のクレジットカードで車両代金を10万ウォン以上お支払いをするお客様
    - 特典 : 50万/30万/20万ウォンからセーブオートご利用額を選択(車種によって利用額に差がございます。)

    [注意事項]
    1. セーブオートとMポイントを合わせて5年内に最大200万ポイントまで使用可能
    2. 現在セーブオートをご利用中のお客様は、既存のセーブオートポイントを返済してから再利用できます。
    3. セーブオート返済期間は、最大36カ月であり、期間中に未返済の残代金は、翌月に一括請求されます。
    4. セーブオート利用期間中に貯まるMポイントは、セーブオートで自動で返済
    30万

生産依頼の予想納期

  • (トゥアラ) ディーゼル : 2カ月、LPI : 4カ月
  • (カーゴ – 3人乗り) ディーゼル : 2カ月、 LPI : 4カ月

一部の購入条件(特典)は、重複適用できません。

車両が出庫センターから出庫され、顧客への引き渡しが行われまで、さらに1~5日かかる場合があります。
