Create New Library Content Page

아래 양식을 작성해 신규 라이브러리 콘텐츠를 업로드 하십시오.
제출이 완료되면 관리자가 검토 후 피드백을 전달해 드립니다.
기본 라이브러리 콘텐츠 내용은 영문으로 입력합니다.

카테고리 구분을 선택해주세요.

  • Brand Campaign
  • Brand Space
  • Brand Collection
  • Global Launching Assets
  • Social Media
  • Event
  • GMA

카테고리 구분을 선택해주세요.

선택

  • Hyundai
  • N
  • Truck & Bus
  • 용량은 최대 20MB 이하까지 가능합니다.
  • 이미지 형식(jpg, png) 파일만 업로드 가능합니다.

기간을 선택해 주세요.

연관 태그를 선택해주세요.

  • 2Avante / Elantra(AD)

연관 태그를 선택해주세요.

  • 한국어
  • 중국어

라이브러리 콘텐츠 입력 가이드

  • Step 1. 라이브러리 콘텐츠 정보 입력 - 파일 등록

    BMS에 업로드할 프로젝트 내용을 입력해주세요. 라이브러리 콘텐츠 정보를 입력하고, BMS에서 사용자에게 소개할 프로젝트 내용으로 구성된 화면을 입력할 수 있습니다. 기본(영어) 콘텐츠를 입력하시면 한국어, 중국어 콘텐츠에 기본 콘텐츠를 복사하여 사용할 수 있습니다.

    • Editor 가이드

      • Editor 상단에 있는 버튼을 클릭하여 텍스트, 이미지, 비디오, 표 등으로 라
        이브러리 콘텐츠를 입력하세요.

      • 폰트 종류, 크기, 스타일, 컬러 설정

      • 텍스트 Align, Bullet, Numbering 설정

      • 이미지, 비디오

      • Editor 확대, HTML 코드뷰

  • Step 2. 파일 상세/저작권 정보 입력

    라이브러리 콘텐츠 정보 입력 시 업로드한 파일의 상세/저작권 정보를 입력해주세요. 검색어로 사용될 파일 제목을 입력하거나 저작권 정보가 동일할 경우 전체 공통 저작권 정보 입력 혹은 개별적으로 저작권 정보가 다를 경우, 파일을 선택하여 정보를 입력할 수 있습니다.

    • 파일 상세/저작권 정보 입력 가이드

      1. 파일 제목

      • 팝업에서 입력하는 파일 제목은 라이브러리 콘텐츠를 검색할 때 검색 결과로 노출될 타이틀입니다. 사용자가 라이브러리 콘텐츠를 검색할 때 검색어로도 사용될 수 있는 정보를 입력해주세요.

      2. 저작권 정보 입력

      • 전체 파일 동일한 정보일 경우 ‘전체 동일 정보 입력’을 선택하고 저작권 정보를 입력하시면 모든 파일의 저작권 정보로 설정됩니다.
      • 파일 별 저작권 정보가 다른 경우 ‘개별 입력’을 선택하고 파일 선택 후, 저작권 정보를 입력하시면 선택한 파일의 저작권 정보로 설정됩니다

  • 리뷰 피드백 프로세스

    리뷰가 정상적으로 요청된 경우 담당자 검토 후 라이브러리 콘텐츠 업로드 여부를 결정합니다.

    • 리뷰 피드백 프로세스

      • 등록하신 리뷰 내용을 담당자가 확인한 후 피드백을 제공합니다.
      • 입력하신 라이브러리 콘텐츠 내용 수정이 필요한 경우 담당자가 수정 요청 피드백을 전달하여 라이브러리 콘텐츠 수정을 진행하게 됩니다.
      • 리뷰 현황은 Review Tools > Library Contents Review > Review Management에서 확인하실 수 있습니다.
      • 리뷰 승인 시, 입력하신 내용은 선택하신 카테고리 구분에 따라 Library > Contents 혹은 Library > Best Practice > GMA 메뉴에 업로드됩니다.

임시저장이 완료되었습니다.
