- 툴팁 팝업

.tooltip-box
.tooltip-box의 상위 요소는 position: relative를 가지고 있어야 함.
.tooltip-box의 상위 요소가 1column일때 .column1 클래스명 부여.
.tooltip-box의 상위 요소가 2column일때 .column2 클래스명 부여.
.tooltip-box의 상위 요소가 5column일때 .column5 클래스명 부여.
