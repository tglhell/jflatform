◎ 폰트 정의

- 언어별 폰트 정의

기본적으로 언어는 "ja"이고, 폰트는 NotoSansJP폰트로 출력되며, 영문은 HyundaiSansHeadOffice, HyundaiSansTextOffice로 출력됨.
본문의 경우는 "HyundaiSansTextOffice"를 사용하기 때문에 클래스 "font-type2"를 사용함.

[NotoSansJP] このテキストは、サンプルのテキストです。


[HyundaiSansHeadOffice] This text is sample text.


[HyundaiSansTextOffice] This text is sample text. This text is sample text. This text is sample text. This text is sample text.


[NotoSansJP, HyundaiSansHeadOffice] このテキストは、123456 This text is sample text. サンプルのテキストです。


[NotoSansJP, HyundaiSansTextOffice] このテキストは、123456 THIS TEXT is sample text. サンプルのテキストです。
