네임 정의

html 파일명 - 기획서상 화면ID 기준

class - 단어와 단어사이에 "-" 하이픈 사용, 숫자는 하이픈 없이 기본두자리수로 작성, 경우에따라 01은 생략할 수 도 있다. EX) btn-area, table-type, table-type02

id - 기본적으로 camelCase사용한다. EX)gnbToggle, searchEvent

image - 언더스코어 "_"로 정의 한다. 숫자는 하이픈 없이 기본두자리수로 작성, 경우에따라 01은 생략할 수 도 있다.
