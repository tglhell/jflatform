BUTTON

기본 클래스인 "btn"에 클래스 추가하여 사용"
컬러 구분- "btn-type2, btn-type3"
사이즈 구분- "btn-middle, btn-small, btn-extra-small"
비활성- ".disabled 또는 disabled="disabled""

◎ Big Button

- .btn / .btn-type2 / .btn-type3

◎ Middle Button

- .btn / .btn-middle

◎ cta-outer

PC: Default / MO: Full or 1n = .cta-outer
PC: Default / MO: Default = .cta-outer + .fix-type
PC/MO: Full or 1n = .cta-outer + .full-type

- .cta-outer

- .cta-outer + .fix-type

- .cta-outer + .full-type

- .cta-outer + .row-type

- width auto

.auto-width 글자수에 따라 버튼의 길이가 달라짐 / min-width 없음

◎ Table Button

클래스 "btn + btn-tbl"
height 36인경우 "btn + btn-tbl + type2 " / PC: max-width: 420px, mo: max-width: 320px;
