◎ Table Button

클래스 "btn"이며, disable 스타일은 클래스 "btn-disabled"를 사용함.
pc에서 가득차는 스타일은 .cta-outer에 클래스 "full-type"을 추가함.
버튼 기본 높이값은 56px, 작은 사이즈는 클래스 "btn-middle, btn-small, btn-extra-small"을 추가함.

◎ Icon Button

기본 클래스인 "btn"에 클래스 "btn-icon"추가하여 사용하고, 아이콘은 css의 ".btn-icon span:before"부분을 수정함.
